Find your Place
Simulador y Analista de Viabilidad Comercial

Una aplicación movil diseñada para ayudar a emprendedores a descubrir la locación ideal para su próximo negocio. Evalúa múltiples zonas geográficas en tiempo real basándose en variables clave como el flujo peatonal, la densidad de competidores y el nivel socioeconómico.

Características Principales:
Inteligencia Artificial Local: Emplea un modelo Random Forest Regressor entrenado e implementado 100% en TypeScript para arrojar puntajes de éxito predictivos de manera offline, mejorando la latencia en el frontend.
Fusión Geoespacial: Combina infraestructura masiva de múltiples proveedores de mapas (OSM/Mapbox) y datos comerciales oficiales (INEGI) en una única pre-evaluación estructurada.
Diversidad Comercial: El algoritmo prioriza la mejor recomendación geográfica mitigando en tiempo real zonas de alta saturación y asegurando diversidad espacial en las sugerencias (Top 5 localizaciones).
Reportes Offline: Funcionalidad nativa de exportación en PDF para facilitar la toma de decisiones.
Tecnologías Principales:
Frontend & App Móvil: Angular 20, Ionic 8, Capacitor 8 (Cross-platform).
Mapas & Geodatos: Mapbox GL JS, OpenStreetMap / Nominatim.
Machine Learning: Algoritmo Random Forest Regressor custom-built (TypeScript).
Generación de Reportes: jsPDF, html2canvas.
